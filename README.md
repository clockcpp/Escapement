# Escapement
A lib for cpp console programs

## how to use it
Just include the file to your program!  
like this:  
```cpp
#include "escapement.h"
```

And don't forget there's a namespace, you need use this
```cpp
esc::writeFileWF(xxx);
```
or this
```cpp
using namespace esc;
```
in any function:
```cpp
writeFileWF(xxx);
```
to use it.
